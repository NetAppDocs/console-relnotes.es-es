=== Actualización de los nombres de las funciones de planificación del ciclo de vida

Las siguientes funciones de planificación del ciclo de vida tienen cambios de nombre:

* La función «Tech Refresh» ahora se llama *System Refresh*. Descubre cómo https://docs.netapp.com/us-en/console-lifecycle-planning/use/tech-refresh.html["evaluar una actualización del sistema"].
* La función «Aumentar capacidad» ahora se llama *Ampliación de almacenamiento*. Descubre cómo https://docs.netapp.com/us-en/console-lifecycle-planning/use/capacity-add.html["ampliar el almacenamiento"].
* La función «Renovaciones» ahora se llama *Renovaciones de soporte*. Descubre cómo https://docs.netapp.com/us-en/console-lifecycle-planning/support/task-support-registration.html["registrarse para soporte"].
